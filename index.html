  • Edgehill NLP Sentiment Analysis

    This project involved applying Natural Language Processing (NLP) techniques to analyse a dataset of video game reviews and determine the sentiment expressed in each review. The program utilises the NLTK library in Python to perform sentiment analysis on individual sentences within the reviews and then aggregates those scores to classify each review as positive or negative. The project also includes an exploration of Named Entity Recognition (NER) to identify and extract key entities mentioned in the reviews.

    Key features:

    • Sentiment analysis using TextBlob to determine the polarity of individual sentences and overall reviews.
    • Algorithm development for classifying reviews as positive or negative based on aggregated sentiment scores.
    • Named Entity Recognition (NER) using NLTK to identify and extract key entities from the text.

    Challenges:

    • Determining an appropriate threshold for classifying reviews based on sentiment scores.
    • Handling inconsistencies and nuances in language to ensure accurate sentiment analysis.

    Lessons learned:

    • The practical application of NLP techniques for analyzing text data and extracting meaningful insights.
    • The importance of considering context and nuances in language when performing sentiment analysis.
    • The value of combining different NLP techniques, such as sentiment analysis and NER, to gain a more comprehensive understanding of text data.
